Краткое содержание

PDF Dark Mode Converter обрабатывает ваши файлы полностью в вашем браузере. Ничего не загружается, не хранится и не передаётся на сервер. Мы не собираем персональные данные.

Обработка файлов

Когда вы используете конвертер, ваш PDF читается JavaScript-кодом, который выполняется локально на вашем устройстве. Преобразование цветов, рендеринг и сборка PDF происходят в вашем браузере. Полученный файл создаётся в оперативной памяти и скачивается прямо на ваш компьютер. Ваш файл ни в какой момент не покидает ваше устройство и не передаётся через интернет.

Аналитика

Этот сайт использует Google Analytics и Cloudflare Web Analytics для понимания общих паттернов трафика, таких как просмотры страниц и количество посетителей. Эти сервисы собирают анонимные, неперсонализированные данные (например, посещённые страницы, приблизительное местоположение по стране, тип браузера). Cloudflare Web Analytics не использует cookies. Google Analytics может использовать first-party cookie для различения посетителей. Ни один из этих сервисов не имеет доступа к вашим PDF-файлам или личной информации. Вы можете отключить Google Analytics, установив дополнение для браузера Google Analytics Opt-out.

Отсутствие сбора персональных данных

Помимо агрегированной статистики, мы не собираем, не храним и не передаём никакие персональные данные. Нет учётных записей, нет сбора email-адресов, нет контактных форм, сохраняющих данные.

Сторонние ресурсы

Сайт загружает open-source JavaScript-библиотеки (pdf.js и pdf-lib) с CDN-сервисов для обеспечения обработки PDF. Эти CDN-провайдеры могут регистрировать стандартную информацию о веб-запросах (IP-адрес, временная метка, запрошенный ресурс) в соответствии со своими политиками конфиденциальности. Никакие персональные данные или содержимое файлов не передаются этим сервисам.

Хостинг

Этот сайт размещён на GitHub Pages. GitHub может собирать стандартные серверные логи, как описано в Политике конфиденциальности GitHub.

Открытый код

Полный исходный код доступен на GitHub. Вы можете проверить любое утверждение на этой странице, прочитав код самостоятельно.

Изменения

Если эта политика будет изменена, дата обновления в верхней части страницы будет обновлена. Поскольку инструмент не собирает данные, изменения маловероятны.

Контакты

Вопросы по этой политике можно задать через GitHub Issues.
