v2.0.0 - 14 апреля 2026

Новый инструмент, более умный конвертер, 14 статей в блоге и полноценный сайт.

Инвертировать цвета PDF

Новый самостоятельный инструмент на /invert-pdf-colors. Выполняет настоящую RGB-инверсию (каждый цвет заменяется комплементарным), а не применяет тему. Тот же GPU-конвейер, та же конфиденциальность, отдельная страница для закладок.

Конвертер

  • Сохранение изображений - включите эту опцию, и конвертер обнаружит фотографии в вашем PDF и оставит их без изменений, затемнив всё остальное. Больше никаких инвертированных портретов.
  • Выделяемый текст - опциональный невидимый текстовый слой для поиска и копирования в сконвертированном PDF.
  • Светлый режим вывода - теперь можно конвертировать из тёмного в светлый, не только из светлого в тёмный. Удобно для печати тёмных PDF на бумаге.
  • Цвета панели инструментов теперь подстраиваются под выбранную тему.

Интерфейс

  • Простой выпадающий список тем заменён на палитру с цветовыми превью каждой темы.
  • Новое цветовое поле с hex-вводом и сеткой предустановленных цветов для пользовательской темы.
  • Настройки (разрешение, качество, переключатели) теперь скрыты за сворачиваемой панелью вместо постоянного отображения.
  • Панель инструментов с эффектом glassmorphism (полупрозрачное размытие).
  • На главной странице появилась карусель блога с прокруткой свайпом.
  • Адаптивный дизайн для смартфонов, планшетов и десктопов.

Контент

  • 14 статей в блоге - руководства по платформам для Chrome, Firefox, Edge, Windows, Android, iPhone, iPad, Google Drive и Acrobat, а также тематические статьи об инверсии vs тёмный режим, печати, ночном чтении и онлайн-конвертации.
  • Весь сайт переведён на 10 языков: английский, испанский, французский, немецкий, итальянский, португальский, китайский, корейский, японский, русский.
  • Страницы «О проекте», «Конфиденциальность», «Условия использования», «Контакты» и этот список изменений.
  • Общий футер на всех страницах.

Исправления ошибок

  • Исправлено обнаружение GPU-фильтров для Firefox (резервный SVG-фильтр canvas теперь работает корректно).
  • Улучшена мобильная верстка: переключатель языка больше не перекрывает навигацию, лента GitHub скрыта на маленьких экранах, адаптивные отступы панели инструментов.
