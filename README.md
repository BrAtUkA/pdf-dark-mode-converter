# PDF Dark Mode Converter

> Originally created by [Chizkiyahu](https://github.com/Chizkiyahu/pdf-dark-mode-converter) · Made up to 10x faster by [BrAtUkA](https://github.com/BrAtUkA)

## Description
PDF Dark Mode Converter is a tool that allows you to convert your PDF files to dark mode. This tool runs completely on your device, ensuring that your privacy is 100% protected.

### What's new in this fork

- **Up to 10x faster** conversion using GPU-accelerated color transforms, JPEG encoding, and concurrent page processing
- **15 themes** including Classic, Slate, Claude Warm, ChatGPT Cool, Solarized, Nord, Dracula, Gruvbox, Monokai, Tokyo Night, Rosé Pine, Cobalt, and more
- **Custom theme** with a color picker and hex input
- **Adjustable resolution and quality** sliders for fine-tuning output
- **Improved UI**

## Instructions
1. Click on the [GitHub Pages link](https://bratuka.github.io/pdf-dark-mode-converter/) to open the tool.
2. Click on the "Choose PDF File" button or drag & drop a PDF onto the page.
3. Optionally adjust the theme, resolution, and quality sliders.
4. Conversion runs automatically. Watch the progress bar with a live time estimate.
5. The converted PDF downloads automatically when done. Click "Download failed? Click here" if needed.
