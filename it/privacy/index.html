Privacy

Ultimo aggiornamento: 13 aprile 2026

Riepilogo

Il Convertitore PDF Modalita Scura elabora i tuoi file interamente nel tuo browser. Nulla viene caricato, memorizzato o trasmesso a un server. Non raccogliamo dati personali.

Elaborazione dei file

Quando usi il convertitore, il tuo PDF viene letto da JavaScript in esecuzione localmente sul tuo dispositivo. La trasformazione dei colori, il rendering e l'assemblaggio del PDF avvengono nel tuo browser. Il file risultante viene creato in memoria e scaricato direttamente sul tuo computer. Il tuo file non lascia mai il tuo dispositivo e non viene trasmesso via internet.

Analisi

Questo sito utilizza Google Analytics e Cloudflare Web Analytics per comprendere i pattern di traffico aggregati come visualizzazioni di pagina e conteggi dei visitatori. Questi servizi raccolgono dati anonimi e non personali (ad es. pagine visitate, posizione approssimativa per paese, tipo di browser). Cloudflare Web Analytics non utilizza cookie. Google Analytics puo utilizzare un cookie di prima parte per distinguere i singoli visitatori. Nessuno di questi servizi ha accesso ai tuoi file PDF o alle tue informazioni personali. Puoi disattivare Google Analytics installando il componente aggiuntivo per la disattivazione di Google Analytics.

Nessuna raccolta di dati personali

A parte le statistiche aggregate, non raccogliamo, memorizziamo o trasmettiamo alcun dato personale. Non ci sono account utente, nessuna raccolta di email e nessun modulo di contatto che registri dati.

Risorse di terze parti

Il sito carica librerie JavaScript open source (pdf.js e pdf-lib) da servizi CDN per abilitare l'elaborazione dei PDF. Questi provider CDN possono registrare informazioni standard di accesso al server web (indirizzo IP, timestamp, risorsa richiesta) secondo le proprie informative sulla privacy. Nessun dato personale o contenuto dei file viene condiviso con questi servizi.

Hosting

Questo sito e ospitato su GitHub Pages. GitHub puo raccogliere log standard del server come descritto nella Informativa sulla privacy di GitHub.

Open source

Il codice sorgente completo e disponibile su GitHub. Puoi verificare qualsiasi affermazione di questa pagina leggendo il codice direttamente.

Modifiche

Se questa informativa dovesse cambiare, la data di aggiornamento in alto a questa pagina verra modificata. Poiche lo strumento non raccoglie dati, le modifiche sono improbabili.

Contatti
