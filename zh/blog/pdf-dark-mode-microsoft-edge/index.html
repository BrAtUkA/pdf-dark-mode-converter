Edge的内置PDF阅读器

Microsoft Edge自带了一个功能相当强大的PDF阅读器。在Edge中打开PDF文件时，它会直接在浏览器标签页中显示，无需插件或扩展。阅读器支持文字选择、搜索、注释和朗读功能。它是浏览器中最好的内置PDF阅读器之一。

然而，Edge的PDF阅读器没有针对PDF内容的「深色模式」按钮。当你将Edge切换为深色模式（「设置」>「外观」>「深色」）时，工具栏和菜单变暗了，但PDF页面保持白色。听起熟悉吗？这和Adobe Acrobat是同样的问题。

方法一：「强制深色模式」标志

Edge有一个实验性功能，可以强制将深色颜色应用到所有网页内容，包括PDF。启用方法如下：

  1. 打开Edge，在地址栏中输入edge://flags。
  2. 搜索「Force Dark Mode for Web Contents」或「Auto Dark Mode for Web Contents」。
  3. 将设置从「Default」改为「Enabled」。
  4. 出现提示时点击重新启动。

重启后，Edge会尝试将深色颜色应用到你打开的每个网页和PDF。对于文字较多的简单PDF，效果还不错，但有以下缺点：

  • 图片会受影响。该标志会反转或改变整个页面的颜色，包括照片、徽标和图形。如果你的PDF包含视觉元素，它们会显得褪色或颜色反转。要更详细地了解这个问题，请阅读我们的深色模式vs颜色反转对比。
  • 效果不一致。有些PDF看起来不错，有些则一塌糊涂。该算法是为网页设计的，而不是PDF文档，因此效果差异很大。
  • 影响所有内容。一旦启用，你访问的每个网页也会被强制进入深色模式。有些网页能正常显示，很多不能。你会不断地开关这个标志。
  • 它是实验性的。Microsoft可能在任何Edge更新中更改、损坏或移除此标志，而不会提前通知。

方法二：Edge的朗读和沉浸式阅读器

对于某些PDF，你可以使用Edge的沉浸式阅读器，它会移除格式并以简洁、可自定义的视图呈现文字。使用方法如下：

  1. 在Edge中打开PDF。
  2. 在地址栏中查找沉浸式阅读器图标（看起来像一本打开的书）。
  3. 如果可用，点击它并将背景颜色更改为深色。

这只适用于Edge能够解析为流动文字的纯文字PDF。扫描文档、复杂布局以及包含分栏或表格的PDF无法使用沉浸式阅读器。当它能用时很有用，但在你最需要的文档上往往不起作用。

常见问题

Edge原生支持PDF深色模式吗？
不是默认支持的。Edge的深色模式设置只影响浏览器界面（工具栏、菜单、标签页），不影响PDF内容区域。「Force Dark Mode」标志是一种实验性解决方案，不是正式功能。

「强制深色模式」标志会影响其他网页吗？
有可能。该标志会将深色颜色转换应用到所有网页内容，而很多网页并非为此设计。你可能在某些网页上遇到可读性问题、不可见的文字或看起来奇怪的图形。

我可以在手机上使用Edge阅读深色PDF吗？
Edge移动版对PDF的支持有限，且没有标志系统。最好的方案是先在电脑（或手机浏览器）上使用转换工具转换PDF，然后阅读转换后的文件。
