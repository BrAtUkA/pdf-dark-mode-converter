Contacto

Convertidor PDF Modo Oscuro es un proyecto gratuito de código abierto mantenido por BrAtUkA. Hay varias formas de ponerse en contacto:

Reportes de errores y solicitudes de funciones

La mejor forma de reportar un error o sugerir una función es abrir un issue en GitHub Issues. Esto mantiene todo organizado y permite que otros usuarios opinen.

Preguntas generales

Para preguntas generales sobre la herramienta, consulta primero las preguntas frecuentes en la página principal. Si tu pregunta no está respondida allí, no dudes en abrir un issue en GitHub o iniciar una Discusión.

Correo electrónico

También puedes contactarnos directamente en bratuka@bratuka.dev.

Contribuir

Los pull requests son bienvenidos. El código fuente está en GitHub. Si planeas un cambio grande, abre un issue primero para discutir el enfoque.
