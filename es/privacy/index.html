Política de Privacidad

Última actualización: 13 de abril de 2026

Resumen

Convertidor PDF Modo Oscuro procesa tus archivos completamente en tu navegador. No se sube, almacena ni envía nada a ningún servidor. No recopilamos datos personales.

Procesamiento de archivos

Cuando usas el convertidor, tu PDF es leído por JavaScript que se ejecuta localmente en tu dispositivo. La transformación de color, el renderizado y el ensamblaje del PDF ocurren en tu navegador. El archivo resultante se genera en memoria y se descarga directamente a tu ordenador. En ningún momento tu archivo sale de tu dispositivo ni viaja por internet.

Analíticas

Este sitio usa Google Analytics y Cloudflare Web Analytics para comprender patrones de tráfico agregados como vistas de página y recuentos de visitantes. Estos servicios recopilan datos anónimos y no personales (por ejemplo, páginas visitadas, ubicación aproximada por país, tipo de navegador). Cloudflare Web Analytics no usa cookies. Google Analytics puede usar una cookie propia para distinguir visitantes únicos. Ninguno de estos servicios tiene acceso a tus archivos PDF ni a información personal. Puedes desactivar Google Analytics instalando el Complemento de Desactivación de Google Analytics.

Sin recopilación de datos personales

Más allá de las analíticas agregadas, no recopilamos, almacenamos ni transmitimos ninguna información personal. No hay cuentas de usuario, no hay recopilación de correo electrónico ni formularios de contacto que almacenen datos.

Recursos de terceros

El sitio carga bibliotecas JavaScript de código abierto (pdf.js y pdf-lib) desde servicios CDN para habilitar el procesamiento de PDF. Estos proveedores de CDN pueden registrar información estándar de acceso al servidor web (dirección IP, marca temporal, recurso solicitado) según sus propias políticas de privacidad. No se comparten datos personales ni contenido de archivos con estos servicios.

Alojamiento

Este sitio está alojado en GitHub Pages. GitHub puede recopilar registros de servidor estándar según se describe en la Declaración de Privacidad de GitHub.

Código abierto

El código fuente completo está disponible en GitHub. Puedes verificar cada afirmación de esta página leyendo el código tú mismo.

Cambios

Si esta política cambia, se revisará la fecha de actualización en la parte superior de esta página. Dado que la herramienta no recopila ningún dato, los cambios son poco probables.

Contacto

Las preguntas sobre esta política pueden dirigirse a GitHub Issues.
