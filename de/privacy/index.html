Datenschutz

Letzte Aktualisierung: 13. April 2026

Zusammenfassung

PDF Dunkelmodus Konverter verarbeitet deine Dateien vollständig in deinem Browser. Es wird nichts hochgeladen, gespeichert oder an einen Server übertragen. Wir erfassen keine personenbezogenen Daten.

Dateiverarbeitung

Wenn du den Konverter verwendest, wird dein PDF durch JavaScript gelesen, das lokal auf deinem Gerät ausgeführt wird. Farbtransformation, Rendering und PDF-Zusammenstellung erfolgen in deinem Browser. Die resultierende Datei wird im Arbeitsspeicher erzeugt und direkt auf deinen Computer heruntergeladen. Deine Datei verlässt zu keinem Zeitpunkt dein Gerät und wird nicht über das Internet übertragen.

Analyse

Diese Website verwendet Google Analytics und Cloudflare Web Analytics, um aggregierte Verkehrsmuster wie Seitenaufrufe und Besucherzahlen zu verstehen. Diese Dienste erfassen anonyme, nicht personenbezogene Daten (z. B. besuchte Seiten, ungefährer Standort nach Land, Browsertyp). Cloudflare Web Analytics verwendet keine Cookies. Google Analytics kann ein Erstanbieter-Cookie verwenden, um einzelne Besucher zu unterscheiden. Keiner dieser Dienste hat Zugriff auf deine PDF-Dateien oder persönliche Informationen. Du kannst Google Analytics deaktivieren, indem du das Google Analytics Opt-out Browser-Add-on installierst.

Keine Erfassung personenbezogener Daten

Abgesehen von aggregierten Statistiken erfassen, speichern oder übermitteln wir keinerlei personenbezogene Daten. Es gibt keine Benutzerkonten, keine E-Mail-Erfassung und keine Kontaktformulare, die Daten speichern.

Drittanbieter-Ressourcen

Die Website lädt Open-Source-JavaScript-Bibliotheken (pdf.js und pdf-lib) von CDN-Diensten, um die PDF-Verarbeitung zu ermöglichen. Diese CDN-Anbieter können standardmäßige Webserver-Zugriffsinformationen (IP-Adresse, Zeitstempel, angeforderte Ressource) gemäß ihren eigenen Datenschutzrichtlinien protokollieren. Es werden keine personenbezogenen Daten oder Dateiinhalte an diese Dienste weitergegeben.

Hosting

Diese Website wird auf GitHub Pages gehostet. GitHub kann Standard-Serverprotokolle erfassen, wie in der GitHub Datenschutzerklärung beschrieben.

Open Source

Der vollständige Quellcode ist auf GitHub verfügbar. Du kannst jede Aussage auf dieser Seite überprüfen, indem du den Code selbst liest.

Änderungen

Sollte sich diese Richtlinie ändern, wird das Aktualisierungsdatum oben auf dieser Seite angepasst. Da das Tool keine Daten erfasst, sind Änderungen unwahrscheinlich.

Kontakt

Fragen zu dieser Richtlinie können über GitHub Issues gestellt werden.
