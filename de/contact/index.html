Fehlerberichte und Funktionswünsche

Am besten meldest du einen Fehler oder schlägst eine Funktion vor, indem du ein Ticket auf GitHub Issues erstellst. So bleibt alles übersichtlich und andere Nutzer können sich einbringen.

Allgemeine Fragen

Für allgemeine Fragen zum Tool schau dir zuerst die FAQ auf der Startseite an. Falls deine Frage dort nicht beantwortet wird, erstelle gerne ein Ticket auf GitHub oder starte eine Diskussion.

E-Mail

Du kannst uns auch direkt unter bratuka@bratuka.dev erreichen.

Mitwirken

Pull Requests sind willkommen. Der Quellcode ist auf GitHub. Wenn du eine größere Änderung planst, erstelle zuerst ein Ticket, um den Ansatz zu besprechen.
