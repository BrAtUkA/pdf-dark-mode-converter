Politique de Confidentialité

Dernière mise à jour : 13 avril 2026

Résumé

Convertisseur PDF Mode Sombre traite vos fichiers entièrement dans votre navigateur. Rien n'est envoyé, stocké ni transmis à un quelconque serveur. Nous ne collectons aucune donnée personnelle.

Traitement des fichiers

Lorsque vous utilisez le convertisseur, votre PDF est lu par du JavaScript exécuté localement sur votre appareil. La transformation des couleurs, le rendu et l'assemblage du PDF se déroulent dans votre navigateur. Le fichier obtenu est généré en mémoire et téléchargé directement sur votre ordinateur. À aucun moment votre fichier ne quitte votre appareil ni ne transite par internet.

Analytique

Ce site utilise Google Analytics et Cloudflare Web Analytics pour comprendre les tendances de trafic agrégées comme les pages vues et le nombre de visiteurs. Ces services collectent des données anonymes et non personnelles (par exemple, pages consultées, localisation approximative par pays, type de navigateur). Cloudflare Web Analytics n'utilise pas de cookies. Google Analytics peut utiliser un cookie propriétaire pour distinguer les visiteurs uniques. Aucun de ces services n'a accès à vos fichiers PDF ni à vos informations personnelles. Vous pouvez désactiver Google Analytics en installant le Module de Désactivation de Google Analytics.

Aucune collecte de données personnelles

En dehors des statistiques agrégées, nous ne collectons, stockons ni transmettons aucune information personnelle. Il n'y a pas de comptes utilisateur, pas de collecte d'adresses e-mail ni de formulaires de contact qui stockent des données.

Ressources tierces

Le site charge des bibliothèques JavaScript open-source (pdf.js et pdf-lib) depuis des services CDN pour permettre le traitement des PDF. Ces fournisseurs de CDN peuvent enregistrer des informations standard d'accès au serveur web (adresse IP, horodatage, ressource demandée) conformément à leurs propres politiques de confidentialité. Aucune donnée personnelle ni contenu de fichier n'est partagé avec ces services.

Hébergement

Ce site est hébergé sur GitHub Pages. GitHub peut collecter des journaux de serveur standard comme décrit dans la Déclaration de Confidentialité de GitHub.

Open-source

Le code source complet est disponible sur GitHub. Vous pouvez vérifier chaque affirmation de cette page en lisant le code vous-même.

Modifications

Si cette politique évolue, la date de mise à jour en haut de cette page sera révisée. Étant donné que l'outil ne collecte aucune donnée, des modifications sont peu probables.

Contact

Les questions relatives à cette politique peuvent être adressées via GitHub Issues.
