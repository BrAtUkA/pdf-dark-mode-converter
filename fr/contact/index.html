Contact

Convertisseur PDF Mode Sombre est un projet gratuit et open-source maintenu par BrAtUkA. Voici les différentes façons de nous contacter :

Signalements de bugs et demandes de fonctionnalités

La meilleure façon de signaler un bug ou de suggérer une fonctionnalité est d'ouvrir un ticket sur GitHub Issues. Cela permet de tout organiser et de laisser les autres utilisateurs donner leur avis.

Questions générales

Pour les questions générales sur l'outil, consultez d'abord la FAQ sur la page d'accueil. Si votre question n'y trouve pas de réponse, n'hésitez pas à ouvrir un ticket sur GitHub ou à lancer une Discussion.

E-mail

Vous pouvez également nous contacter directement à bratuka@bratuka.dev.

Contribuer

Les pull requests sont les bienvenues. Le code source est sur GitHub. Si vous prévoyez un changement important, ouvrez d'abord un ticket pour discuter de l'approche.
