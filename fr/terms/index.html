Ce que fait l'outil

Convertisseur PDF Mode Sombre est une application web côté client qui modifie le schéma de couleurs des documents PDF. Elle fonctionne entièrement dans votre navigateur. Aucun fichier n'est envoyé vers un serveur.

Utilisez les fichiers convertis à votre propre discrétion.
