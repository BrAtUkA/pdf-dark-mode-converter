Contact

PDF Dark Mode Converter is a free, open-source project maintained by BrAtUkA. There are a few ways to get in touch:

Bug reports and feature requests

The best way to report a bug or suggest a feature is to open an issue on GitHub Issues. This keeps everything organized and lets other users chime in.

General questions

For general questions about the tool, check the FAQ on the homepage first. If your question is not answered there, feel free to open a GitHub issue or start a Discussion.

Email

You can also reach out directly at bratuka@bratuka.dev.

Contributing

Pull requests are welcome. The source code is on GitHub. If you are planning a large change, open an issue first to discuss the approach.
