사용 방법

PDF 업로드

색상이 반전됩니다

모든 픽셀이 GPU 가속 색상 매트릭스 변환을 통해 RGB 보색으로 매핑됩니다. 페이지는 속도를 위해 병렬 처리됩니다.

다운로드

반전된 PDF가 자동으로 다운로드됩니다. 보이지 않는 텍스트 레이어가 포함되어 텍스트를 선택, 복사, 검색할 수 있습니다.

PDF 색상 반전이란?

색상 반전은 PDF의 모든 색상을 RGB 보색으로 대체합니다. 흰색 배경은 검정으로, 검정 텍스트는 흰색으로, 다른 모든 색상은 스펙트럼의 반대편으로 전환됩니다. 결과는 원본 문서의 실제 사진 네거티브입니다.

화면 표시만 변경하는 OS 접근성 필터나 브라우저 확장 프로그램과 달리, 이 도구는 반전된 색상이 영구적으로 포함된 새 PDF 파일을 생성합니다. 공유하고, 인쇄하고, 다른 기기에서 열어도 색상은 반전된 상태를 유지합니다.

이 도구를 사용하는 이유

영구 반전

색상이 실제로 반전된 PDF를 생성합니다. 화면 필터가 아니라 파일 자체가 변경됩니다.

GPU 가속

GPU에서 SVG 색상 매트릭스 필터를 사용하고 페이지를 병렬 처리합니다. 50페이지도 몇 초 만에 완료.

선택 가능한 텍스트

반전된 PDF는 보이지 않는 텍스트 레이어를 유지하여 검색, 선택, 복사가 가능합니다.

완전한 프라이버시

파일이 기기를 절대 떠나지 않습니다. 업로드 없음, 서버 없음, 클라우드 없음. 오프라인으로 작동합니다.

스캔된 PDF 지원

픽셀 수준에서 반전하므로 텍스트 기반과 이미지 기반 PDF 모두 작동합니다.

조절 가능한 품질

해상도와 JPEG 품질 슬라이더로 파일 크기와 선명도를 조절할 수 있습니다.

반전 vs. 다크 모드

실제 색상 반전은 모든 색상을 수학적 보색으로 변환합니다. 흰색은 순수 검정으로, 파란색은 주황색으로, 녹색은 마젠타로. 이미지와 컬러 그래픽에도 영향을 미치는 네거티브 효과를 생성합니다.

다크 모드 변환은 더 스마트합니다: 흰색 배경을 특정 어두운 색조로 변환하면서 이미지 색상을 유지하고 텍스트 가독성을 보장합니다. 더 정교한 읽기 경험을 원하신다면 전체 변환기에서 Dracula, Nord, Solarized 등 16가지 이상의 테마 다크 모드를 제공합니다.
