PDF 다크 모드 변환기 ← 블로그

개인정보 처리방침

최종 업데이트: 2026년 4월 13일

개요

PDF 다크 모드 변환기는 브라우저에서 완전히 파일을 처리합니다. 서버로 전송, 저장, 전송되는 것은 아무것도 없습니다. 개인 데이터를 수집하지 않습니다.

파일 처리

변환기를 사용하면 PDF는 기기에서 로컬로 실행되는 JavaScript로 읽힙니다. 색상 변환, 렌더링, PDF 조립이 모두 브라우저에서 이루어집니다. 생성된 파일은 메모리에서 만들어져 컴퓨터로 직접 다운로드됩니다. 파일이 기기를 떠나거나 인터넷을 통해 전송되는 일은 없습니다.

분석

이 웹사이트는 페이지 조회수, 방문자 수 등 집계 트래픽 패턴을 파악하기 위해 Google Analytics와 Cloudflare Web Analytics를 사용합니다. 이 서비스들은 익명의 비개인 데이터(방문 페이지, 대략적인 국가 위치, 브라우저 유형)를 수집합니다. Cloudflare Web Analytics는 쿠키를 사용하지 않습니다. Google Analytics는 방문자 구분을 위해 퍼스트파티 쿠키를 사용할 수 있습니다. 어떤 서비스도 PDF 파일이나 개인 정보에 접근할 수 없습니다. Google Analytics 차단 브라우저 추가 기능을 설치하여 Google Analytics를 거부할 수 있습니다.

개인 데이터 미수집

위에 언급된 집계 통계 외에 개인 데이터를 수집, 저장, 전송하지 않습니다. 사용자 계정 없음, 이메일 수집 없음, 데이터를 기록하는 문의 양식 없음.

타사 리소스

웹사이트는 PDF 처리를 위해 CDN 서비스에서 오픈소스 JavaScript 라이브러리(pdf.js, pdf-lib)를 로드합니다. 이러한 CDN 제공업체는 자체 개인정보 처리방침에 따라 표준 웹 서버 접근 정보(IP 주소, 타임스탬프, 요청 리소스)를 기록할 수 있습니다. 이러한 서비스와 개인 데이터나 파일 내용이 공유되지 않습니다.

호스팅

이 웹사이트는 GitHub Pages에서 호스팅됩니다. GitHub는 GitHub 개인정보 보호 정책에 따라 표준 서버 로그를 수집할 수 있습니다.

오픈소스

전체 소스 코드는 GitHub에 공개되어 있습니다. 이 페이지의 내용을 코드를 직접 읽어 확인할 수 있습니다.

변경 사항

이 방침이 변경되면 페이지 상단의 업데이트 날짜가 변경됩니다. 이 도구는 데이터를 수집하지 않으므로 변경 가능성은 매우 낮습니다.
