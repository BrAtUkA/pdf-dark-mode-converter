v2.0.0 - 2026년 4월 14일

새 도구, 더 스마트한 변환기, 14개 블로그 글, 완전한 웹사이트.

PDF 색상 반전

/invert-pdf-colors에 위치한 새로운 독립 도구. 테마를 적용하는 대신 실제 RGB 반전(각 색상을 보색에 매핑)을 수행합니다. 동일한 GPU 파이프라인, 동일한 프라이버시 보호, 전용 페이지로 즐겨찾기 가능.

변환기

  • 이미지 유지 - 이 옵션을 켜면 변환기가 PDF 내 사진을 감지하고 나머지를 어둡게 변환하면서 원래 색상을 유지합니다. 반전된 인물 사진이 더 이상 나타나지 않습니다.
  • 선택 가능한 텍스트 - 선택적 보이지 않는 텍스트 레이어로 변환된 PDF에서 검색 및 복사가 가능합니다.
  • 라이트 모드 출력 - 이제 밝은 색에서 어두운 색뿐만 아니라 어두운 색에서 밝은 색으로도 변환 가능합니다. 어두운 PDF 인쇄에 적합합니다.
  • 툴바 색상이 선택한 테마에 맞게 자동 조정됩니다.

인터페이스

  • 단순 테마 드롭다운이 색상 미리보기가 포함된 전용 테마 선택기로 업그레이드.
  • 커스텀 색상 옵션을 위한 새 색상 패널, HEX 입력 및 프리셋 색상 그리드 포함.
  • 설정(해상도, 품질, 토글)이 펼침 가능한 패널로 이동, 항상 표시되지 않음.
  • 툴바에 글래스모피즘(반투명 블러) 효과 적용.
  • 메인 페이지에 터치 스와이프 지원 블로그 캐러셀 추가.
  • 모바일, 태블릿, 데스크톱에 적응하는 반응형 디자인.

콘텐츠

  • 14개 블로그 글 - Chrome, Firefox, Edge, Windows, Android, iPhone, iPad, Google Drive, Acrobat에 대한 플랫폼 가이드와 반전 vs. 다크 모드, 인쇄, 야간 읽기, 온라인 변환 주제 글.
  • 전체 사이트가 10개 언어로 번역됨: 영어, 스페인어, 프랑스어, 독일어, 이탈리아어, 포르투갈어, 중국어, 한국어, 일본어, 러시아어.
  • 소개, 개인정보, 이용약관, 문의, 변경 이력 페이지.
  • 모든 페이지에 통일된 푸터.

버그 수정

  • Firefox용 GPU 필터 감지 수정 (캔버스 SVG 필터 폴백이 이제 올바르게 작동합니다).
  • 모바일 레이아웃 개선: 언어 선택기가 더 이상 내비게이션과 겹치지 않으며, 작은 화면에서 GitHub 리본이 숨겨지고, 반응형 도구 모음 간격이 적용됩니다.
