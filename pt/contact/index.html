Contato

O Conversor PDF Modo Escuro e um projeto gratuito e open source mantido por BrAtUkA. Veja como entrar em contato:

Relatos de bugs e sugestoes de funcionalidades

A melhor forma de relatar um bug ou sugerir uma funcionalidade e abrir um ticket no GitHub Issues. Assim tudo fica organizado e outros usuarios podem contribuir.

Perguntas gerais

Para perguntas gerais sobre a ferramenta, consulte primeiro as FAQ na pagina principal. Se sua pergunta nao for respondida la, abra um ticket no GitHub ou inicie uma discussao.

Email

Voce tambem pode entrar em contato diretamente pelo email bratuka@bratuka.dev.

Contribuir

Pull requests sao bem-vindos. O codigo-fonte esta no GitHub. Se voce esta planejando uma alteracao grande, abra um ticket primeiro para discutir a abordagem.
