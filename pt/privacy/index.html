Privacidade

Ultima atualizacao: 13 de abril de 2026

Resumo

O Conversor PDF Modo Escuro processa seus arquivos inteiramente no seu navegador. Nada e enviado, armazenado ou transmitido a um servidor. Nao coletamos dados pessoais.

Processamento de arquivos

Quando voce usa o conversor, seu PDF e lido por JavaScript executando localmente no seu dispositivo. A transformacao de cores, renderizacao e montagem do PDF acontecem no seu navegador. O arquivo resultante e criado na memoria e baixado diretamente no seu computador. Seu arquivo nunca sai do seu dispositivo e nao e transmitido pela internet.

Analise

Este site usa Google Analytics e Cloudflare Web Analytics para entender padroes de trafego agregados como visualizacoes de pagina e contagens de visitantes. Esses servicos coletam dados anonimos e nao pessoais (ex.: paginas visitadas, localizacao aproximada por pais, tipo de navegador). O Cloudflare Web Analytics nao usa cookies. O Google Analytics pode usar um cookie de primeira parte para distinguir visitantes individuais. Nenhum desses servicos tem acesso aos seus arquivos PDF ou informacoes pessoais. Voce pode desativar o Google Analytics instalando o complemento de desativacao do Google Analytics.

Sem coleta de dados pessoais

Alem das estatisticas agregadas, nao coletamos, armazenamos ou transmitimos nenhum dado pessoal. Nao ha contas de usuario, nenhuma coleta de email e nenhum formulario de contato que registre dados.

Recursos de terceiros

O site carrega bibliotecas JavaScript open source (pdf.js e pdf-lib) de servicos CDN para habilitar o processamento de PDF. Esses provedores de CDN podem registrar informacoes padrao de acesso ao servidor web (endereco IP, timestamp, recurso solicitado) conforme suas proprias politicas de privacidade. Nenhum dado pessoal ou conteudo de arquivo e compartilhado com esses servicos.

Hospedagem

Este site e hospedado no GitHub Pages. O GitHub pode coletar logs padrao do servidor conforme descrito na Declaracao de Privacidade do GitHub.

Open source

O codigo-fonte completo esta disponivel no GitHub. Voce pode verificar qualquer afirmacao desta pagina lendo o codigo diretamente.

Alteracoes

Se esta politica mudar, a data de atualizacao no topo desta pagina sera alterada. Como a ferramenta nao coleta dados, alteracoes sao improvaveis.

Contato

Perguntas sobre esta politica podem ser feitas pelo GitHub Issues.
